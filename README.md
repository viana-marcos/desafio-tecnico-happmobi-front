# DesafioTecnicoHappmobiFront


## Development server


Para testar essa aplicação instale a versão v22.21.1 do nodejs e  siga os passos abaixo para execução.
A aplica já cria na inicialização no backend o usuário admin@gmail.com com a senha 123, no caso do usuário locador acesse o formulário de adastro na tela de login(url: http://localhost:4200/login) clicando no link "Criar Conta" 

## Project setup

```bash
$ npm install
```

## Para rodar a aplicação execute o comando abaixo

```bash
# development
$ npm run start
